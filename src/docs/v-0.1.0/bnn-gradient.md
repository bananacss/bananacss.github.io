# bnn-gradient

*Banana code:*
```css
.demo {
  bnn-gradient: #f9e400 #ff9c00 vertical;
}
```

*Result:*
```css
.demo {
  background-image: linear-gradient(
                    to bottom,
                    #f9e400,
                    #ff9c00
  );
}
```
