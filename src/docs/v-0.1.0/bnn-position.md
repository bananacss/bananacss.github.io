# bnn-position

*Banana code:*
```css
.demo {
  bnn-position: 10px 5px 8px 90px;
}
```

*Result:*
```css
.demo {
  top: 10px;
  right: 5px;
  bottom: 8px;
  left: 90px;
}
```

*Banana code:*
```css
.demo {
  bnn-position: center;
}
```

*Result:*
```css
.demo {
  display: block;
  margin-left: auto;
  margin-right: auto;
}
```
